Headlines / Page Titles

Headlines are often a different color than standard headings and can be assigned to any H1-H6 tag using class="headline". To create a highlight, place <strong> tags around the words to be highlighted.

Headline with a highlight.

<h1 class="headline">Headline with a <strong>highlight</strong>.</h1>

Sub-Headings

Add a sub-heading to any H1-H6 tag using <span class="subTitle"> tag.

Heading Text This is a sub-heading

<h2>Heading Text<span class="subTitle">This is a sub-heading</span></h2>

Blockquote

This is the default blockquote style applied to this theme's <blockquote> tags. Each skin has a custom color for the blockquote style. You can also create your own custom styles to match any skin using the following code in your stylesheet:
blockquote {
border-color: [your color];
}
